# 🚀 Internship Projects

This repository contains the projects completed during my internship. These projects demonstrate my skills in **Python**, **Cybersecurity**, **Secure Software Development**, and **Web Application Security**.

---

## 📂 Projects Included

### 🔹 1. Web Application Vulnerability Scanner

A Python-based web application that scans websites for common security vulnerabilities and generates detailed security reports.

#### ✨ Features

* 🔍 Website Crawling
* 🛡️ SQL Injection Detection
* ⚠️ Cross-Site Scripting (XSS) Detection
* 📋 Security Header Analysis
* 📊 Scan History Dashboard
* 📄 PDF Report Generation
* 📑 CSV Export
* 💾 Scan History Storage

#### 🛠️ Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Requests
* BeautifulSoup

---

### 🔹 2. Secure Image Steganography Tool

A Python application that securely hides and extracts secret messages inside images using steganography and encryption.

#### ✨ Features

* 🖼️ Hide Secret Messages in Images
* 🔓 Extract Hidden Messages
* 🔐 Encryption for Secure Communication
* 🖥️ User-Friendly GUI
* 📁 File-Based Encryption & Decryption

#### 🛠️ Technologies Used

* Python
* OpenCV
* Cryptography
* NumPy
* Tkinter
